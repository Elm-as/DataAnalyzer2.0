Votre parcours en 7 étapes

1. Import des Données

Importez une ou plusieurs bases de données (CSV, Excel, JSON). Travaillez sur plusieurs bases simultanément si nécessaire.

2. Aperçu des Données

Prévisualisez les premières lignes et colonnes pour vérifier le format et l'importation correcte de vos données.

3. Rapport de Qualité

Analyse complète : valeurs manquantes, doublons, colonnes problématiques. Recevez des avertissements et suggestions personnalisés.

4. Configuration

Sélectionnez votre variable cible, vérifiez les types de données, et choisissez les features pertinentes pour l'analyse.

5. Options d'Analyse

Choisissez parmi les analyses de base (statistiques, corrélations) et avancées (ML, Deep Learning). Activez uniquement ce qui est possible.

6. Résultats

Visualisez tous les résultats par onglets : métriques détaillées, graphiques, et un résumé complet des analyses effectuées.

7. Simulation & Prédiction

Utilisez le meilleur modèle pour faire des prédictions sur de nouvelles données. Visualisez les probabilités et contributions de chaque variable.

Fonctionnalités clés
  • Gestion intelligente des corrélations : Identifiez et gérez les features trop corrélées pour éviter le surentraînement
  • Analyses conditionnelles : Seules les analyses compatibles avec vos données sont proposées
  • Métriques complètes : Toutes les métriques professionnelles pour chaque type d'analyse
  • Export complet : Rapports PDF/HTML, code Python reproductible, modèles entraînés
